Новости / 5 новых настроек таргетинга

Доброго времени суток!

1. Обновились настройки таргетинга при создании заданий, добавлено 5 новых параметров:

— Семейное положение. Параметр, который требуется, как правило, свадебной тематике, но найдет применение и в любой другой. Имеется возможность выбора нескольких СП.

— Количество фотографий профиля. Параметр, с помощью которого можно исключить, к примеру, исполнителей с малым числом фотографий.

— Количество записей на стене. Данный параметр поможет исключить исполнителей с пустой стеной, либо наоборот, исполнителей, на чьих стенах большое количество записей (как правило, рекламных).

— Дата регистрации страницы. Важный параметр, с помощью которого вы сможете определить, страницы с каким возрастом могут выполнять ваши задания. Подойдет для тех, кто хочет исключить в исполнителях своих заданий, например, страницы со свежей датой регистрации.

— Временной таргетинг. С помощью этого параметра можно настроить требуемое количество выполнений задания в сутки, которое автоматически распределится по часам.

2. Обновился раздел "Мои задания", новый функционал включает в себя:

— Расширенный поиск. Форма, позволяющая фильтровать задания по ссылке, типу кампании, статусу и другим параметрам.

— Название заданий. Для удобства ориентирования по заданиям, любому заданию можно задать название.

— Информер, показывающий количество заданий по статусам.

Мы открыты к предложениям по улучшению сервиса, будем рады выслушать их в комментариях к этой новости.
